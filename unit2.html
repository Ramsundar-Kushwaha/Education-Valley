Unit 2

Classes in C++

  • A class is a user-defined data type.
  • It groups data members (variables) and member functions (methods).
  • Acts as a blueprint for creating objects.

Class Declaration

  • Declares a class using the class keyword.
  • Defines structure but doesn't allocate memory until objects are created.
  • Syntax:
  • class ClassName {// data members and member functions}

Declaring Objects

  • An object is an instance of a class.
  • Created using the class name.
  • Example:
  • ClassName obj1, obj2;

Defining Member Functions

  • Functions that operate on class data.
  • Can be defined inside or outside the class.
  • Outside class syntax:
  • void ClassName :: functionName() { // code }

Inline Member Function

  • Defined inside the class.
  • Treated as inline, reducing function call overhead.
Example
class A {
  void show() { cout << "Hello";} // Inline
};

Array of Objects

  • An array that stores multiple objects of a class.
  • Example:
  • ClassName obj[5]; // array of 5 objects

Objects as Function Argument

  • You can pass objects by value, by reference, or by pointer to functions.
  • Example:
  • void show(Student s); // by value
  • void display(Student &s); // by reference

Static Data Member and Member Function

  • Static members are shared by all objects.
  • Belong to the class, not to individual objects.
  • Syntax:
  • class A { static int count;
  • static void show(); };
